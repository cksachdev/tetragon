nameIDproperty 
nameID:String

A personal name for the unit. This is optional and can be used if a unit should have a recognizable name among it's type of units, for example a spy unit could have a personal name to make it easier to be recognized among other spy units. If no name ID was specified for the unit, this property will return the same as the unit's designation ID.
