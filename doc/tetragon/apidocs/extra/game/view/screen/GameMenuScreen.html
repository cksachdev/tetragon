addChildren()method
override protected function addChildren():void

Used to add all display children to the display list of the screen. By default this method will automatically add all registered displays to the display list in the same order they were registered. You can override this method if you want to add the display children manually.

addListeners()method 
override protected function addListeners():void

Used to add any event or signal listeners to child objects of the display.

This is an abstract method. Override this method and add any listeners to objects that require event/signal listening.

createChildren()method 
override protected function createChildren():void

Used to create all display children that are part of the screen. Called right after the screen has been created.

This is an abstract method. Override it in your sub-screen class and instanciate any display child objects here that are part of the screen.

layoutChildren()method 
override protected function layoutChildren():void

Used to layout the display children of the display. This method is called initially to set the position and size of any child objects and should be called whenever the children need to update their position or size because the layout has changed, for example after the application window has been resized.

This is an abstract method. Override it in your sub-display class and set the position and size of all child display objects here.

registerDisplays()method 
override protected function registerDisplays():void

Registers displays for use with the screen.

This is an abstract method. Override this method in your sub-screen class and register the displays by using the registerDisplay() method. Displays that are registered with the screen have most of their methods called automatically when these methods are called on the screen. These methods are: init(), start(), stop(), reset(), update(), dispose(), enabled and paused.

removeListeners()method 
override protected function removeListeners():void

Used to remove any event or signal listeners from child objects that were added inside the addListeners() method. Called automatically when the display is being disposed.

This is an abstract method. Override this method and remove any event/signal listeners here that were added in addListeners().
