Packageextra.game.render.tile
Classpublic class TileArea
InheritanceTileArea Object

TileArea represents an area on a tile map that the tile engine uses to decide which tile groups are currently visible and which are not. The tile engine divides a tile map into areas where every area's size is equal to the size of the tile scroller's view port size. The TileArea class keeps track of which tile groups on it are currently visible.
