Classpublic class BaseEffect
InheritanceBaseEffect Object

A base effect is an object that can be used by an ability, an item or virtually any other entity as the effect that the associated object causes. A base effect consists of one or more base effect definitions which describe the type of effect, on what the effect has an influence, etc.
