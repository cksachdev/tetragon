Packagebase.command
Classpublic class Command
InheritanceCommand Object
Subclasses CLICommand, PausableCommand

Abstract class for command implementations. A command encapsulates code that can be instantiated and executed anywhere else in the application.

Commands can either be executed without signal listening if their execution code is processed synchronous or a command can be listened to for signals that are broadcasted by the command when the it completes, an error occurs or during command progress steps.

complete()method 
protected function complete():void

Completes the command. This is an abstract method that needs to be overridden by subclasses. You put code here that should be executed when the command finishes, like cleaning up event listeners etc. After your code, place a call to super.complete().
