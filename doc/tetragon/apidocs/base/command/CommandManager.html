Packagebase.command
Classpublic class CommandManager
InheritanceCommandManager Object
Implements ICommandListener

A manager that can be used to manage command execution. You call the execute method and specify a command and any handler methods that should be notified of broadcasted command events. After the command has finished execution all it's event listeners are automatically removed. The CommandManager also makes sure that the same command is not executed more than once at the same time.
