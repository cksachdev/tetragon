The Preloader preloads the web-based application SWF which it is part of. This class becomes the root of the SWF by delegation in the entry class. After preloading has finished, the Entry class is instantiated inside this class and is notified that the preload process has been finished (by calling onApplicationPreloaded).

This class acts as a wrapper for various different preload display classes. The default used implementation is TetragonPreloadDisplay. You can write your own preload displays by implementing the IPreloadDisplay interface and then configure your class for use with this preloader inside the configure() method.
