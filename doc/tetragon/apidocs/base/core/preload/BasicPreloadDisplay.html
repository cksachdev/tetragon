Classpublic class BasicPreloadDisplay
InheritanceBasicPreloadDisplay Object
Implements IPreloadDisplay

A very basic implementation of IPreloadDisplay. The BasicPreloadDisplay only waits until the application is fully loaded and doesn't show any indication about the load progress.
