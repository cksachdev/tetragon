_skipDelayproperty 
protected var _skipDelay:int = 10

How many frames to wait before initiating the preloader. This is a threshold used to wait for several frames before displaying the preloader bar and text in case the SWF is already in the cache or locally loaded in which case the preloader doesn't need to appear. Setting this value to 0 will always display the preloader.
