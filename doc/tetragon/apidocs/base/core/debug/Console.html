Packagebase.core.debug
Classpublic final class Console
InheritanceConsole flash.display.Sprite

A class that represents a Debugging and Command output console similar to that found in many later games. By default the console - once instantiated and added to the stage - is hidden and can be toggled visible with the toggle() method.
