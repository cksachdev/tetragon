Packagebase.core.settings
Classpublic class LocalSettingsManager
InheritanceLocalSettingsManager Object

This is a class that manages LocalSharedObject settings. You can use this class to store and recall persistent data to the users harddisk into a local shared object.

To store local settings with the LocalSettingsManager you first create a LocalSettings object and put all settings values into it that need to be stored. Then you use the store() method to store the settings to disk.
