Packagebase.io.resource.loaders
Classpublic class ResourceLoader
InheritanceResourceLoader Object
Subclasses BinaryResourceLoader, ImageResourceLoader, SoundResourceLoader, SWFResourceLoader, XMLResourceLoader

The base class for resource loaders. A resource loader is used by the resource manager and resource providers to load resources through the AS3 API that supports loading them correctly. Depending on the loaded resource file the method how it is loaded differs.

For example the implementation for loading a sound file is different than that for an image file. The resource loaders take care that those files are loaded the way they need to be loaded. In other words: A resource loader essentially represents the basic resource file type.

Resource loaders can load files from the filesystem as well as embedded files.
