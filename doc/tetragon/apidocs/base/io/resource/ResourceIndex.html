addResource()method 
public function addResource(id:String, path:String, packageID:String, dataFileID:String, resourceFileClass:Class, family:String, type:String, embedded:Boolean = false):void

Adds a resource to the resource index. Note that a resource is not a concrete file resource but a data object that contains information about where to find the resource file, it's data type, whether it's embedded or not, etc. and the actual file data, once it has been loaded.

Parameters

id:String — The unique ID of the resource.
 
path:String — The path of the resource file, either on the filesystem or inside a resource package file.
 
packageID:String — The package ID if the resource is packed into a package.
 
dataFileID:String — The ID of the datafile in which this resource can be found. Media and textdata resources have no dataFileID.
 
resourceFileClass:Class — The resource class of the resource.
 
family:String — The resource family of the resource.
 
type:String — The data type of the resource (if it's a data or entity resource).
 
embedded:Boolean (default = false) — Whether the resource is embedded or not.
