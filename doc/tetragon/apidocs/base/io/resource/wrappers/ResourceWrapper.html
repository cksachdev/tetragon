Subclasses BinaryResourceWrapper, ImageResourceWrapper, SoundResourceWrapper, SWFResourceWrapper, XMLResourceWrapper

The base class for resource wrappers. A resource wrapper is used by the resource manager and resource providers to temporarily wrap resource files that are being loaded. Depending on the loaded resource file the method how it is loaded differs.

For example the implementation for loading a sound file is different than that for an image file. The resource wrappers take care that those files are loaded the way they need to be loaded. In other words: A resource wrapper essentially represents the resource file type.

Resource wrappers can load files from the filesystem as well as embedded files.
