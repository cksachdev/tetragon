Classpublic class Image32ResourceWrapper
InheritanceImage32ResourceWrapper ImageResourceWrapper ResourceWrapper flash.events.EventDispatcher

This is a ResourceFile subclass for image data. It allows you to load an image file format supported by Flash (JPG, PNG, or GIF) and access it as a BitmapData or Bitmap. Image32ResourceFile provides a 32bit bitmap.
