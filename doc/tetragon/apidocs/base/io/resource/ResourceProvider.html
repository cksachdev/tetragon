checkBulkComplete()method 
protected function checkBulkComplete(bulkFile:ResourceBulkFile):void

Checks whether all resources in the current bulk have been processed. It's not guaranteed which occurs first: all resources have been processed or the loader has completed. So to make sure that the current loaded bulk is finished and ready for use we wait until both the resources and the loader is complete.
