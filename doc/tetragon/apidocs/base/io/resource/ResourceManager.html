load()method 
public function load(resourceIDs:*, completeHandler:Function = null, loadedHandler:Function = null, failedHandler:Function = null, progressHandler:Function = null, forceReload:Boolean = false):void

Loads one or more resources either from the file system, from a packed resource file or from embedded files.

Parameters

resourceIDs:* — Can be one of the following: a string containing one resource ID, an array with one or more resource IDs or a queue with one or more resource IDs.
 
completeHandler:Function (default = null) — An optional callback handler that is called after all resources have completed the load procedure, regardless if some or all of them failed or succeeded to load.
 
loadedHandler:Function (default = null) — An optional callback handler that is called everytime a resource has been loaded.
 
failedHandler:Function (default = null) — An optional callback handler that is called everytime a resource failed to load.
 
progressHandler:Function (default = null) — If true the resource manager will reload the resource again from it's file, even if it already has been loaded or failed.
 
forceReload:Boolean (default = false)

unload()method 
public function unload(resourceIDs:*):void

Unloads previously loaded resources. This does not necessarily mean the resource will be available for garbage collection. Resources are reference counted so if the specified resource has been loaded multiple times, its reference count will only decrease as a result of this.

Parameters

resourceIDs:* — IDs of the resources to unload.
