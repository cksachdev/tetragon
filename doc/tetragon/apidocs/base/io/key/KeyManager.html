assign()method 
public function assign(keyValue:*, mode:int, callback:Function, ... params):KeyCombination

Assigns a keyboard key or key combination to a callback function. The specified keyValue can be either a String, a uint, a KeyCombination object or an Array.

By specifying a String you can assign a key combination which determines that the exact keys in this combination need to be pressed to trigger the callback. The String can contain one or more key identifiers which are divided by the KEY_COMBINATION_DELIMITER constant (a plus sign '+'), for example F1, CTRL+A or SHIFT+CTRL+1.

By specifying a uint you can assign one key by it's key code directly. With this you can also use the numeric contants from ActionScript's native Keyboard class.

By specifying a KeyCombination object the key manager assigns the KeyCombination object with the key codes that are already listed in the KeyCombination.

By specifying an Array you can assign multiple key combinations to the same callback. The Array can contain a combination of String key identifiers, uints and/or KeyCombination objects.

Parameters

keyValue:* — The key value to assign. This can be one of the following object types: String, uint, KeyCombination or Array.
 
mode:int — The mode of the key assignment that determines when the callback is triggered, either when the key is pressed or released or whether it is repeatedly triggered or a sequence of keys that need to be entered to trigger the callback. You can use the KeyMode class with it's constants.
 
callback:Function — The method that is called when the key or key combination is triggered.
 
... params — A list of optional parameters that are provided as arguments to the callback function.

Returns
KeyCombination — A KeyCombination object or null. If the assignment succeeded the resulting KeyCombination object is returned, if the assignment failed null is returned. If the specified keyValue is an Array only the last successful assignment from the arrays' containing key values is returned. if all assignments from the array failed, null is returned.

createKeyCombination()method 
public static function createKeyCombination(keyString:String, type:int):KeyCombination

Creates a key-combination object from a string that defines either one or multiple keys identifier strings, a key code or a key sequence.

Parameters

keyString:String — A string that defines a key identifier or multiple key identifiers, separated by the KeyManager.KEY_COMBINATION_DELIMITER (+), e.g. CTRL+C.
 
type:int — The type of the specified string, can be either TYPE_KEYSTRING, TYPE_KEYCODE or TYPE_KEYSEQ.

Returns
KeyCombination — A KeyCombination object or null.

remove()method 
public function remove(keyValue:*, mode:int):KeyCombination

Allows to remove a specific key assignment from the key manager.

Parameters

keyValue:* — The key value to remove. This can be either a String, a uint a KeyCombination object or an Array containing any of these.
 
mode:int (default = NaN) — The key mode of the to-be-removed key assignment.

Returns
KeyCombination — The KeyCombination object that was removed or null if no key mapping was found with the specified keyValue and mode.
