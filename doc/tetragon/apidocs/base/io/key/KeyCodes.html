getKeyCode()method
public static function getKeyCode(keyName:String):int

Returns the keycode for the specified key name if it is found in the key codes table. A key name is a standard identifier used to identify keys or key combinations (e.g. 'page_up'). The character case is irrelevant here, i.e. PAGE_UP is the same as page_up.

Parameters

keyName:String — the name of the key for that the key code should be returned.

Returns
int — the key code or -1 if the key name was not found in the code table.
