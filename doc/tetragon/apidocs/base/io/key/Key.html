Packagebase.io.key
Classpublic class Key
InheritanceKey flash.events.EventDispatcher

A Singleton Key class that simplifies listening to global key strokes and adds additional keyboard events. Key enables you to receive events when multiple keys are held/released and when a sequence of keys is pressed.

getKeyCodes()method 
public static function getKeyCodes(keyString:String):Array

Returns an array of key codes that are created from the specified keyString. The keyString can contain one or more key names combined by the key combination delimiter (+), e.g. CTRL+SHIFT+A or just CTRL. If no valid key names or key combination was found it returns null.

Parameters

keyString:String — the string of keys to provide key codes from.

Returns
Array — an array comprising of the key codes or null.
isCombination()method 
public static function isCombination(keyString:String):Boolean

Checks whether the specified key string contains a valid combination of keys. A valid key combination string is comprised of at least two valid key names (see KeyCodes class) which are connected by a plus (+) character, e.g. CTRL+C. A key combination can contain several keys, e.g. CTRL+SHIFT+F1. A string that only contains a single key (e.g. CTRL) is not considered a key combination.

Parameters

keyString:String — The string to check for key codes.

Returns
Boolean — true if the string contains two or more connected keycodes.
