Packagebase.io.file.loaders
Classpublic class ConfigLoader
InheritanceConfigLoader FileLoader flash.events.EventDispatcher

A class that loads the application configuration/ini file and parses the loaded properties into the config model. The manager/config model supports simple string and numeric values, objects, and arrays. The Config model should contain all properties that are also found in the config file. See the Config class for more info.
