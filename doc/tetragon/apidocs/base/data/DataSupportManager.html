DataSupportManagerMethods
 
Packagebase.data
Classpublic class DataSupportManager
InheritanceDataSupportManager Object

An odd multi-role class that manages the mapping and creation of several objects used by the resource management and the entity system. It acts as an index for parser-, entity component- and complex type classes by mapping data parser classes to datatype IDs and entity component classes to component class IDs.
