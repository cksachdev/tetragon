Classpublic class LoadProgressDisplay
InheritanceLoadProgressDisplay flash.display.Sprite

A simple progress bar that displays the load progress of resources being loaded for the displays of a screen. The LoadProgressDisplay itself is not of type Display and cannot load resources for itself. It's only purpose is to show load progress.
