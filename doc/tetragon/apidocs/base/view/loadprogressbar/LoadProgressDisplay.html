Property Detail
closeScreenBeforeLoadproperty
closeScreenBeforeLoad:Boolean  [read-only]

Determines if the load progress display should request to close any screen that might be open. By default this property returns false which means that the load progress display appears overlaid over any open screen. You can override this getter in your sub-class and let it return true to have any open screen closed before the load progress display is shown.

The default value is false.

waitForUserInputproperty 
waitForUserInput:Boolean  [read-only]

Determines whether the load progress display should wait for user input before continuing after the loading completed. You can set this property to return true in your sub-class if you want the progress display to wait for user input after loading has finished.

The default value is false.
